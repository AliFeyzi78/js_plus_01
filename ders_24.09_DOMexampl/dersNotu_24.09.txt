JS'den HTML' ye erişim için;

- getElement
1. id ile
    document.getElementById("xxxxxx");
2. class ile:
    const element = document.getElementsByClassName("xxxxxx");
3. tag ile:
    const element = document.getElementsByTagName("xxxxxx");

- querySelector
1. id veya class için "#" veya "." ile
    const element = document.querySelector("#xxxxx");

- querySelectorAll
1. bütün id veya class'lar için "#" veya "." ile ()
    const element = document.querySelectorAll(".xxxxxxx");
2. tag'ler için
    const element = document.querySelectorAll("p");

bir element'in attributunu değiştirmek

- get ve set attribute
    const element = document.querySelector("#link"); -->ilgili tag'e ulaşıyoruz.

    const attr = element.getAttribute("href"); --> sonra attribute'a ulaşıyoruz.
    console.log(attr);

    element.setAttribute("href", "https://clarusway.com");  --> attribute'u değiştiriyoruz.
